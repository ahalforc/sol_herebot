# sol_herebot

Sol Herebot is the champion of OSRS Discord Bots!

## Tech stack

### Bun

This project was created with `bun` via `bun init` v1.1.29. See [Bun](https://bun.sh) for details.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

To upgrade Bun:

```bash
bun upgrade
```
